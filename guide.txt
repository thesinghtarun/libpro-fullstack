        flutter pub get(make sure developer mode is enabled)
    -Open frontend(in project folder)
        -lib
            -consts.dart

    -Open backend(in terminal)

            -npm install

    -open frontend
        -cd..
            -cd frontend
                -flutter run
                    -1(for windows)
                    -2(for edge)
                    -3(for chrome)
